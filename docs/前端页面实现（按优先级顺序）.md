# 前端页面实现（按优先级顺序）

> 本文档是 Phase 4 的详细补充。执行 prompt 中 Phase 4 只需每页写一句话占位；**后续要实现具体页面内容时，按本文档逐页执行**。
> 所有组件样式参考 `docs/design-system.html`，设计令牌参考 `docs/frontend-style-prompt.md`。

---

## P0：登录 + 仪表盘 + 用户管理（首屏必需）

### 1. 登录页 `/login`

- 新建 `frontend/src/app/login/page.tsx`
- 全屏居中布局：`min-h-screen flex items-center justify-center bg-[var(--color-bg)]`
- 卡片容器（420px 宽）：`<Card>` + `shadow-clay-lg` + `border-3`
- 卡片内部：
  - 品牌区（居中）：48px 蓝紫方块图标（`bg-primary border-3 border-[var(--color-text)] rounded-[var(--radius-sm)]`），下方标题"登录 microDify"（24px/700），副标题"使用你的团队账号登录 AI Agent 平台"（14px/次要色）
  - 邮箱输入框：`<Input>` label="邮箱地址"，placeholder="admin@microdify.local"
  - 密码输入框：`<Input type="password">` label="密码"
  - 行内 flex：`<Checkbox>` "记住我" + "忘记密码？"链接（右对齐）
  - 登录按钮：`<Button variant="primary">` 全宽
  - 分割线（"或"）+ "还没有账号？注册新账号"链接
- 登录逻辑：调 `POST /auth/login`，失败 Toast 报错，成功跳转 `/`

### 2. 仪表盘 `/`

- 重写 `frontend/src/app/page.tsx`
- 统计卡片行：`grid grid-cols-[repeat(auto-fill,minmax(220px,1fr))] gap-5`
  - 6 张 `<Card>` 统计卡片，每张 `<Card.Body>` 包含：
    - `stat-label`：13px 次要色文字（"活跃对话应用"/"Agent 运行中"/"知识库"/"今日对话数"/"平均响应时间"/"Token 用量"）
    - `stat-value`：28px font-mono 粗体蓝紫色数字
    - `stat-change`：12px 趋势文字，绿色箭头 up / 红色箭头 down
- 双列布局：`grid grid-cols-2 gap-5`
  - 左列 `<Card>`：快捷操作
    - `<Card.Header>`："快捷操作"
    - `<Card.Body>`：5 个按钮 flex-wrap（新建对话应用 / 创建 Agent / 上传知识库 / 新建工作流 / 写 Prompt 模板），primary + secondary 混排
  - 右列 `<Card>`：最近活动
    - `<Card.Header>`："最近活动"
    - `<Card.Body>`：`<ul>` 时间线列表，每项 `flex items-center gap-3`，彩色圆点（chat=蓝紫 / agent=薄荷绿 / knowledge=暖橙），标题文字 + 右侧时间戳（12px 三级色）

### 3. 用户管理 `/users`

- 新建 `frontend/src/app/users/page.tsx`
- 顶部工具栏：`flex items-center justify-between`
  - 左侧：搜索 Input（260px）+ 状态筛选自定义 Select（140px）
  - 右侧："+ 新建用户" primary 按钮
- 用户列表 `<Table>`：
  - 列：用户（头像圆圈+用户名）/ 邮箱 / 角色 Tag / 状态 Tag / 创建时间 / 操作
  - 头像：32px 圆角方块 `bg-primary` 或 `bg-secondary` 或 `bg-accent`，白字首字母，`border-2 border-[var(--color-text)]`
  - 操作：编辑（ghost btn）+ 禁用（ghost btn 红色文字）
- 新建用户内联表单（点"+ 新建用户"展开）：`<Card>` 520px 宽
  - `<Card.Header>`："创建新用户"
  - `<Card.Body>`：用户名 Input + 邮箱 Input + 密码 Input + 角色自定义 Select
  - `<Card.Footer>`：取消 secondary 按钮 + 创建 primary 按钮

## P1：核心业务页面

### 4. 对话列表 `/chat`

- 重写 `frontend/src/app/chat/page.tsx`
- 标题"对话应用" + "新建对话应用" primary 按钮
- 卡片网格：`grid grid-cols-[repeat(auto-fill,minmax(300px,1fr))] gap-5`
- 每张 `<Card>`：
  - 48px 图标方块（`bg-primary-light border-3 border-text`）+ 名称 + 描述（14px 次要色，两行截断）
  - Tag 行：模型标签 + 知识库数标签
  - "进入对话 →" 链接（primary 色，粗体）

### 5. 对话详情 `/chat/[id]`

- 重写 `frontend/src/app/chat/[id]/page.tsx`
- 对话气泡区：`max-w-[720px] mx-auto`
  - 用户气泡：`flex flex-row-reverse`，头像 36px 蓝紫方块，气泡 `bg-primary-light border-3 border-text rounded-[var(--radius-md)] rounded-br-[4px]`
  - AI 气泡：`flex`，头像 36px 薄荷绿方块，气泡 `bg-surface border-3 border-text shadow-clay-sm rounded-[var(--radius-md)] rounded-bl-[4px]`
  - 流式生成中：气泡内文字末尾闪烁光标（`w-[2px] h-4 bg-primary animate-pulse inline-block`）+ 气泡上方"正在思考..."斜体 12px 文字
  - 知识库引用：气泡下方小卡片，`bg-bg border-2 border-border rounded-sm p-2 text-xs`，文件图标 + "引用自: **文档名** · 第X节"
- 底部输入区：`<Input>` + 发送按钮，`sticky bottom-0`

### 6. Agent 列表 `/agent`

- 重写 `frontend/src/app/agent/page.tsx`
- 标题"Agent 管理" + "创建 Agent" primary 按钮
- 卡片网格：`grid grid-cols-[repeat(auto-fill,minmax(300px,1fr))] gap-5`
- 每张 `<Card>`：
  - 顶部 flex：48px 图标方块 + 名称 + 状态指示灯
  - 状态灯：8px 圆点，在线=success 绿 / 离线=tertiary 灰，前有文字标签
  - 描述（14px 次要色）
  - Tag 行：模型标签（primary）+ 工具数标签（default）+ 知识库数标签（info）

### 7. Agent 编辑 `/agent/[id]`

- 重写 `frontend/src/app/agent/[id]/page.tsx`
- `<Card>` 表单，max-w-[640px]：
  - 名称 Input
  - 系统指令 Textarea（等宽字体，200px 高）
  - LLM 模型 自定义 Select
  - 启用工具：`<Checkbox>` 组（知识库检索 / 代码执行 / 网页搜索 / 图片生成）
  - 关联知识库：多选列表
  - 模型温度：`<Radio>` 组（精确 0.1 / 平衡 0.7 / 创造 1.0）
  - 保存 primary 按钮 + 取消 secondary 按钮

### 8. 知识库列表 `/knowledge`

- 重写 `frontend/src/app/knowledge/page.tsx`
- 标题"知识库" + "新建知识库" primary 按钮
- `<Table>`：
  - 列：名称 / 类型 Tag / 状态 Tag / 文档数（font-mono）/ 更新时间 / 操作
  - 操作：编辑 + 删除（ghost 红色）

### 9. 知识库详情 `/knowledge/[id]`

- 重写 `frontend/src/app/knowledge/[id]/page.tsx`
- 顶部：知识库名称 + 描述 + 统计（文档数/分块数/向量维度 font-mono 大数字）
- 上传区：虚线边框拖拽区域（`border-3 border-dashed border-border rounded-lg p-8 text-center`），hover 变蓝紫边框
- 文档 `<Table>`：文件名 / 类型 Tag / 分块数 / 状态 / 上传时间 / 操作
- 状态 Tag 示例：已发布（success）/ 处理中（warning）/ 解析失败（error）

### 10. 工作流列表 `/workflow`

- 重写 `frontend/src/app/workflow/page.tsx`
- 标题"工作流" + "新建工作流" primary 按钮
- 卡片网格：每张卡片显示名称、节点数、状态、更新时间

### 11. 工作流编辑器 `/workflow/[id]`

- 重写 `frontend/src/app/workflow/[id]/page.tsx`
- 预留画布容器：`<Card>` 内 `min-h-[500px]`，居中显示"工作流画布（待实现）"占位文字 + 图标
- 画布容器 border-3 + shadow-clay-md + rounded-lg + bg-bg

### 12. Prompt 列表 `/prompt`

- 重写 `frontend/src/app/prompt/page.tsx`
- 标题"Prompt 模板" + "新建模板" primary 按钮
- `<Table>`：模板名称 / 变量数（font-mono）/ 创建时间 / 操作

### 13. Prompt 编辑器 `/prompt/[id]`

- 新建 `frontend/src/app/prompt/[id]/page.tsx`
- 双栏布局：`grid grid-cols-[1fr_320px] gap-5`（移动端单栏）
- 左栏：名称 Input + 模板内容 Textarea（等宽字体，min-h-[200px]，含 `{{variable}}` 占位符示例）
- 右栏：
  - 变量面板 `<Card>`：自动列出所有 `{{变量名}}`，每行变量名（font-mono 蓝紫色）+ 类型 Tag（文本/上下文/RAG）+ 必填/自动注入标注
  - 实时预览 `<Card>`：将已赋值的变量高亮替换（`bg-primary-light text-primary font-mono font-semibold`）入模板原文
